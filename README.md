# 🌍 Barambe — Creative Lab Project

## 🧠 Overview

**Barambe** est un projet créatif issu d’un **Creative Lab**, mêlant **design, innovation et narration digitale**.  
Il explore la création d’une expérience immersive combinant identité visuelle, storytelling et développement web.

🔗 **Projet en ligne :**  
[https://barambe.netlify.app/](https://barambe.netlify.app/)

📄 **Dossier complet :**  
[https://guillaumegoizet.netlify.app/Dossier%20Cr%C3%A9ative%20Lab%20-%20Barambe%201.pdf](https://guillaumegoizet.netlify.app/Dossier%20Cr%C3%A9ative%20Lab%20-%20Barambe%201.pdf)

---

## 🎯 Objectifs

- Développer un **concept créatif fort** autour de l’identité Barambe  
- Concevoir une **expérience utilisateur immersive**  
- Mettre en œuvre une **direction artistique cohérente**  
- Créer un **prototype web interactif**  
- Explorer les méthodologies de **Creative Lab / design thinking**

---

## 🧩 Concept

Barambe s’inscrit dans une démarche de :  

- **Narration visuelle**  
- **Exploration culturelle et artistique**  
- **Création d’univers digital**

Le projet vise à traduire une identité en une expérience interactive, en croisant :  

- Design graphique  
- Motion / animation  
- Développement web  
- Storytelling  

---

## 🛠️ Technologies utilisées

- **HTML / CSS / JavaScript**  
- **Frameworks front-end** (ex : React ou équivalent)  
- **Netlify** pour le déploiement  
- **Outils de design** (Figma, Adobe Suite…)

---

## 🎨 Direction artistique

La direction artistique repose sur :  

- Une **identité visuelle forte**  
- Un travail sur la **typographie et les couleurs**  
- Une **approche immersive centrée sur l’utilisateur**  
- Une cohérence entre **fond (concept)** et **forme (interface)**  

---

## 🚀 Fonctionnalités

- Interface interactive  
- Navigation immersive  
- Expérience storytelling  
- Prototype web accessible en ligne  

---

## 🧪 Méthodologie

Le projet suit une approche **Creative Lab**, basée sur :  

- Collaboration interdisciplinaire  
- Expérimentation rapide  
- Prototypage  
- Itérations successives  

Ces méthodes favorisent l’innovation et la co-création.
